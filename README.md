# Excel---Passengers_Flight_Booking_Analysis

An Excel-based data analysis project with 600 rows of passenger flight booking data.
The dashboard is created based on this dataset using pivot tables to visualize key insights.

## 📊 Key Features
- 600 rows of structured passenger flight booking data
- Data cleaning and preparation in Excel
- Pivot tables for data-driven analysis
- Dashboard built from the dataset to track booking trends and flight status

## 🛠 Tool Used
- Microsoft Excel

## 📁 Contents
- Excel file containing the dataset
- Pivot tables and dashboard based on the data
